BeeGeek Dictionary
На дом Тимуру задали выучить слова и их описание с толкового словаря BeeGeek Dictionary. Мама поручила младшему сыну Артуру проверить брата. Для проверки Артур называет Тимуру слова из словаря — а он должен воспроизводить их описания. Каждая запись в словаре — это слово и текстовое описание его значения.
Помогите Тимуру справиться с проверкой.
Входные данные
В первой строке задаётся целое число N (1≤N≤1000) — количество записей в толковом словаре BeeGeek Dictionary.
В каждой из следующих N строк дано по одной записи: сначала идёт слово, а затем через пробел непустое описание его значения. Все слова, значение которых записаны в словаре, различны.
В следующей строчке после толкового словаря записано целое число M (1≤M≤100)— количество слов, которое проверит Артур.
В следующих M строках перечислены слова для проверки, по одному на строке. Слова записаны в точности так, как они представлены в словаре.
Выходные данные
Для каждого слова из списка Артура выведите описание его значения из словаря или фразу Нет в словаре, если такого слова нет в словаре.

Sample Input:

3
ЗАДАЧА То, что дано или необходимо разрешить, выполнить
BeeGeek лучшая онлайн школа
РАБОТА ПРАЗДНОСТЬ
3
BeeGeek
ПРАЗДНОСТЬ
ЗАДАЧА

Sample Output:

лучшая онлайн школа
Нет в словаре
То, что дано или необходимо разрешить, выполнить
