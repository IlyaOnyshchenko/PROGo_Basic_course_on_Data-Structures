Оценки Тимура 
Преподаватель по программированию в школе BeeGeek хочет выставить годовую оценку Тимуру. Он заметил очень странную закономерность. Оценок у Тимура четное количество, а именно 2⋅N, но, кроме этого, он заметил, что у Тимура N+1 уникальных оценок и ровно одна из этих оценок повторяется N раз. Он решил, что вот именно эту оценку и поставит Тимуру. Помогите преподавателю выставить оценку Тимуру. 
Входные данные
В первой строке задаётся целое число N (2≤N≤10000).
В следующей строке через пробел записаны 2⋅N оценок Тимура. Оценка принимает значения от 0 до 10000.
Выходные данные
Выведите оценку Тимура, которую поставит преподаватель. 

Sample Input:

4
1 2 3 4 5 3 3 3

Sample Output:

3
