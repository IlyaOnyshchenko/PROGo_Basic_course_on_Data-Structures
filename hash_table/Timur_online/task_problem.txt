Тимур на связи
Тимур решил записать телефоны всех своих друзей, чтобы потом автоматизировать поиск нужного номера.
У каждого из друзей Тимура может быть один или больше номеров телефонов. Напишите программу, которая поможет Тимуру находить все номера определённого друга.
Входные данные
В первой строке задано одно целое число N (1≤N≤1000) — количество номеров телефонов, информацию о которых Тимур решил сохранить в телефонной книге.
В следующих N строках заданы телефоны и имена их владельцев через пробел. Телефон — это несколько цифр, записанных подряд, имя же состоит только из русских букв. Записи не повторяются.
В следующей строке записано целое число M (1≤M≤100) — количество запросов от Тимура.
В следующих M строках записаны сами запросы, по одному на строке. Каждый запрос — это имя какого-то друга, чьи телефоны Тимур хочет сейчас найти, записанное в точности так, как в телефонной книге.
Выходные данные
Для каждого запроса от Тимура выведите в отдельной строке все телефоны, принадлежащие человеку с этим именем. Телефоны одного человека выводите в одну строку через пробел в том порядке, в котором они были заданы во входных данных. Количество строк в ответе должно быть равно количеству запросов Тимура.
Если в телефонной книге нет телефонов человека с таким именем, выведите в соответствующей строке Имени {имя} нет в телефонной книге, где вместо {имя} вставляете конкретное имя друга.

Sample Input:

4
988871142 Иосиф
462486 Артур
777999 Иосиф
740015 Леван
4
Артур
артур
Иосиф
Леван

Sample Output:

462486
Имени артур нет в телефонной книге
988871142 777999
740015
