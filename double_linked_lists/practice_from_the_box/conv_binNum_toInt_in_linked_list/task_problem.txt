Преобразование двоичного числа в связанном списке в целое число
Дан список из n цифр, который представляет из себя двоичное представление числа. Значение каждого элемента равно либо 0 или 1. По данному списку вычислите десятичное значение представленного числа. 
Входные данные
Cтрока из n цифр (0 или 1), например 0 0 0 1 0 1
Выходные данные
Выведите десятичное значение представленного двоичного числа. 

Sample Input:

1 0 0

Sample Output:

4
