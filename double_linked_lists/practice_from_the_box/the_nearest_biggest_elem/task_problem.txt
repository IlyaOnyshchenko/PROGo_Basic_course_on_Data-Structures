Ближайший бОльший элемент
Дан список A из n целых положительных чисел.
Для каждого элемента Ai найдите ближайшее бОльшее значение, то есть для каждого Ai нужно найти такое Aj, что j>i и Aj>Ai и j минимально возможное. Если такого j не существует, то будем считать ближайшее большее значение равным 0.
Говоря простыми словами, для каждого элемента нужно найти ближайший справа элемент, который больше него. Если такого не существует, считать 0 ближайшим элементом. 
 
Входные данные
В первой строке вводятся n положительных чисел, каждое из которых не превосходит 10^9
Выходные данные
Выведите на одной строке через пробел ближайший бОльший элемент для каждого элемента. 

Sample Input 1:

1 3 5 2 4

Sample Output 1:

3 5 0 4 0

Sample Input 2:

2 1 5

Sample Output 2:

5 5 0

Sample Input 3:

5

Sample Output 3:

0

Sample Input 4:

10 12

Sample Output 4:

12 0
