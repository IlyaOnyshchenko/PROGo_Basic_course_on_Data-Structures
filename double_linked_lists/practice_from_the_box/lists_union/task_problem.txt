Объединение списков
Даны два упорядоченных по не убыванию списка A и B. Объедините их в один упорядоченный по не убыванию список C. То есть результирующий список должен содержать A.Count()+B.Count() элементов. 
Входные данные
В первой строке задается n целых чисел списка A, упорядоченных по не убыванию, каждое из которых по модулю не превосходит 10^5. 
Во второй строке вводятся m целых чисел списка B, упорядоченных по не убыванию, каждое из которых по модулю не превосходит 10^5. 
Выходные данные
Программа должна вывести последовательность неубывающих чисел, полученный объединением двух списков.

Sample Input 1:

1 2 4
2 3 4

Sample Output 1:

1 2 2 3 4 4

Sample Input 2:

1 2 3
4 5 6

Sample Output 2:

1 2 3 4 5 6
