Удаление дубликатов
При наличии отсортированного связанного списка удалите все дубликаты, чтобы каждый элемент появлялся только один раз.
 Входные данные
В первой строке вводятся n целых отсортированных по не убыванию чисел, каждое из которых по модулю не превосходит 100
Выходные данные
Выведите измененный список без дубликатов на одной строке через пробел.

Sample Input:

1 2 3 3 4

Sample Output:

1 2 3 4
