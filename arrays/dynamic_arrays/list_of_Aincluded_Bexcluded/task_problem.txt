Формирование списка
Даны два списка целых чисел A и B. Сформируйте новый список C, состоящий из элементов, входящих в A, но не входящих в B.
Входные данные
В первой строке задается одно натуральное число n, не превосходящее 100.
Во второй строке вводятся n целых чисел списка A, каждое из которых по модулю не превосходит 100. 
В третьей строке задается одно натуральное число m, не превосходящее 100.
В четвертой строке вводятся m целых чисел списка B, каждое из которых по модулю не превосходит 100. 
Выходные данные
Программа должна вывести список C, состоящий из элементов, входящих в A, но не входящих в B. Порядок элементов должен сохраниться.

Sample Input 1:

2
8 11
1
8

Sample Output 1:

11

Sample Input 2:

3
8 -11 5
1
-11

Sample Output 2:

8 5
