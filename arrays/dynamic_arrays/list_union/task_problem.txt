Объединение списков
Даны два упорядоченных по не убыванию списка A и B. Объедините их в один упорядоченный по не убыванию список C. То есть результирующий список должен содержать len(A)+len(B) элементов. 
len – возвращает количество элементов среза. 
Входные данные
В первой строке задается одно натуральное число n, не превосходящее 10^5.
Во второй строке вводятся n целых чисел списка A, упорядоченных по не убыванию, каждое из которых по модулю не превосходит 10^5. 
В третьей строке задается одно натуральное число m, не превосходящее 10^5.
В четвертой строке вводятся m целых чисел списка B, упорядоченных по не убыванию, каждое из которых по модулю не превосходит 10^5. 
Выходные данные
Программа должна вывести последовательность неубывающих чисел, полученный объединением двух списков.

Sample Input 1:

3
1 2 4
3
2 3 4

Sample Output 1:

1 2 2 3 4 4

Sample Input 2:

3
1 2 3
3
4 5 6

Sample Output 2:

1 2 3 4 5 6
