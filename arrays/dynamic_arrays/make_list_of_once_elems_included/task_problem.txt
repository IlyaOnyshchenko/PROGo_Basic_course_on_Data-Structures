Формирование списка 2
Дан список целых чисел A. Необходимо в списке A оставить лишь те элементы, которые встречаются единожды. 
Входные данные
В первой строке задается одно натуральное число n, не превосходящее 100.
Во второй строке вводятся n целых чисел списка A, каждое из которых по модулю не превосходит 100. 
Выходные данные
Программа должна вывести список, состоящий из элементов, которые встречаются единожды. Порядок элементов должен сохраниться.

Sample Input 1:

3
8 1 1

Sample Output 1:

8

Sample Input 2:

6
8 1 1 3 8 -8

Sample Output 2:

3 -8
