Меньше среднего арифметического
Дан массив из 5 целых чисел. Необходимо найти количество элементов, значение которых меньше среднего арифметического, взятого от всех элементов массива.
Входные данные
Строка из 5 целых чисел, каждое из которых по модулю не превосходит 100. 
Выходные данные
Выведите количество элементов, значение которых меньше среднего арифметического, взятого от всех элементов массива.

Sample Input 1:

1 2 3 4 5

Sample Output 1:

2

Sample Input 2:

-2 -1 0 1 2

Sample Output 2:

2
