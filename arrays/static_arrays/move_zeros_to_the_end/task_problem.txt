Передвижение нулей
Дан массив из 5 целых чисел. Переместите все нули в конец, сохраняя относительный порядок ненулевых элементов. Вы должны сделать это в том же массиве, не создавая нового массива.
 Входные данные
Строка из 5 целых различный чисел, каждое из которых по модулю не превосходит 100. 
Выходные данные
Переместите все нули в конец, сохраняя относительный порядок ненулевых элементов и выведите измененный массив.

Sample Input 1:

1 0 2 0 3

Sample Output 1:

1 2 3 0 0

Sample Input 2:

0 0 1 2 3

Sample Output 2:

1 2 3 0 0

Sample Input 3:

1 3 2 0 0

Sample Output 3:

1 3 2 0 0
