Сумма четных положительных элементов
Дан массив из 5 целых чисел. Необходимо найти сумму четных положительных элементов.
Входные данные
Строка из 5 целых чисел, каждое из которых по модулю не превосходит 100100. 
Выходные данные
Выведите сумму четных положительных элементов.

Sample Input 1:

1 -1 2 -2 3

Sample Output 1:

2

Sample Input 2:

2 -2 2 -2 3

Sample Output 2:

4
