Сумма цифр массива
Дан массив из 3 целых чисел. Необходимо найти сумму всех цифр массива. Например, если дан массив [25,36,78], то сумма всех его цифр будет равна 2+5+3+6+7+8=31.
Входные данные
Строка из 3 целых чисел, каждое из которых по модулю не превосходит 100. 
Выходные данные
Выведите сумму всех цифр массива.
 
Примечание: знак числа на цифры не влияет.

Sample Input 1:

25 36 78

Sample Output 1:

31

Sample Input 2:

-1 -2 -3

Sample Output 2:

6
