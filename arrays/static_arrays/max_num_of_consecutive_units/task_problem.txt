Максимальное количество последовательных единиц
Дан массив из 5 бинарных чисел (либо 0, либо 1). Найдите максимальное количество подряд идущих единиц.
Входные данные
Строка из 5 чисел, каждое из которых принимает значения 0 или 1. 
Выходные данные
Выведите максимальное количество подряд идущих единиц.

Sample Input 1:

0 1 1 0 1

Sample Output 1:

2

Sample Input 2:

1 1 1 1 0

Sample Output 2:

4

Sample Input 3:

1 0 1 1 1

Sample Output 3:

3
