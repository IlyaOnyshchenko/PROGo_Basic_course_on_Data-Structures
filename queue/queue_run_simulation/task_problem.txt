Очередь
Реализуйте структуру данных "очередь". Напишите программу, содержащую описание очереди и моделирующую работу очереди, реализовав все указанные здесь методы. Программа считывает последовательность команд и в зависимости от команды выполняет ту или иную операцию. После выполнения каждой команды программа должна вывести одну строчку.
Возможные команды для программы:
•	enqueue n - Добавить в очередь число n (значение n задается после команды). Программа должна вывести ok. 
•	dequeue - Удалить из очереди первый элемент. Программа должна вывести его значение. 
•	peek - Программа должна вывести значение первого элемента, не удаляя его из очереди. 
•	count - Программа должна вывести количество элементов в очереди. 
•	clear - Программа должна очистить очередь и вывести ok. 
•	exit - Программа должна вывести bye и завершить работу.
 
Входные данные
Вводятся команды управления очередью, по одной на строке. 
Гарантируется, что набор входных команд удовлетворяет следующим требованиям: максимальное количество элементов в очереди в любой момент не превосходит 100, все команды dequeue и peek корректны, то есть при их исполнении в очереди содержится хотя бы один элемент.
Выходные данные
Требуется вывести протокол работы с очередью, по одному сообщению на строке

Sample Input:

enqueue 1
peek
exit
Sample Output:
ok
1
bye
