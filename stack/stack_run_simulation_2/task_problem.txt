Стек 2
Напишите программу, моделирующую работу стека, реализовав все указанные в условии методы. Программа считывает последовательность команд и в зависимости от команды выполняет ту или иную операцию. После выполнения каждой команды программа должна вывести одну строчку.
Возможные команды для программы:
•	push n - Добавить в стек число n (значение n задается после команды). Программа должна вывести ok.
•	pop - Удалить из стека последний элемент. Программа должна вывести его значение.
•	peek - Программа должна вывести значение последнего элемента, не удаляя его из стека.
•	count - Программа должна вывести количество элементов в стеке.
•	clear - Программа должна очистить стек и вывести ok.
•	exit - Программа должна вывести bye и завершить работу.
Перед исполнением операций peek и pop программа должна проверять, содержится ли в стеке хотя бы один элемент. Если во входных данных встречается операция peek или pop, и при этом стек пуст, то программа должна вместо числового значения вывести строку error.
Входные данные
Команды управления стеком вводятся в описанном ранее формате по 1 на строке.
Выходные данные
Требуется вывести протокол работы со стеком, по 1 сообщению в строке

Sample Input:

push 1
peek
exit

Sample Output:

ok
1
bye
