39 школа
Директор школы №39 хочет узнать, сколько однофамильцев учатся в его школе. У него есть список фамилий, и на основании этого списка нужно вычислить количество фамилий, которые совпадают с другими.
Входные данные
В первой строке записано целое число N (1≤N≤100) – количество учеников школы №39.
Затем идут N строк с фамилиями учеников в произвольном порядке.  Все строки содержат символы латинского алфавита и цифры. Длина каждой строки не превышает 100. 
 
Выходные данные
Выведите количество однофамильцев в школе.

Sample Input 1:

6
Dzeranov
Guev
Tigiev
Guev
Dzeranov
Guev

Sample Output 1:

5

Sample Input 2:

3
Dzeranov
Guev
Tigiev

Sample Output 2:

0
