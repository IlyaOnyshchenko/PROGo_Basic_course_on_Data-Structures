Школа
Каждый ученик в классе изучает либо французский, либо немецкий, либо оба этих языка.
У классного руководителя есть списки учеников, изучающих каждый из языков. Напишите программу, которая позволит классному руководителю быстро выяснить, сколько учеников изучает только один язык.
Входные данные
В первой строке записано целое число N (1≤N≤100)– количество учеников, изучающих французский язык.
Затем идут N строк с фамилиями учеников, которые изучают французский язык. Все строки содержат символы латинского алфавита, и длина каждого из них не превышает 100. 
Затем записано целое число M (1≤M≤100)– количество учеников, изучающих немецкий язык.
Затем идут MM строк с фамилиями учеников, которые изучают немецкий язык. Все строки содержат символы латинского алфавита, и длина каждого из них не превышает 100. 
Гарантируется, что среди учеников нет однофамильцев.
Выходные данные
Выведите количество учеников, которые изучают только один язык. Если таких не окажется выведите NO.

Sample Input 1:

3
Ivanov
Petrov
Sidorov
2
Ivanov
Dzeranov

Sample Output 1:

3

Sample Input 2:

3
Ivanov
Petrov
Dzeranov
3
Ivanov
Petrov
Dzeranov

Sample Output 2:

NO
